# Calculator CLI App

##  Overview
This is a simple command-line calculator built in Python as part of a developer internship task.  
The app supports basic operations like addition, subtraction, multiplication, and division.

##  Features
- User-friendly CLI interaction
- Supports `+`, `-`, `*`, `/`
- Handles invalid inputs & division-by-zero
- Runs continuously until the user types `exit`
